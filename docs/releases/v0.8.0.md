# v0.8.0 Release Notes

`v0.8.0 — Pre-v1.0 production hardening`

---

## Overview

v0.8.0 stabilizes configuration validation, production defaults, public API inventory, and the **observability contract** (metric names, label policy, hook semantics, OTEL attributes, debug snapshot versioning). Most v0.7 callers require no code changes.

---

## Observability contract (KL-1804)

### Stable metrics (no renames)

Prometheus adapter metric names are unchanged from v0.7. The canonical inventory is `keylane.StableMetricDescriptors()` (27 families). Contract tests ensure the prometheus collector matches the registry.

### Label policy

- Stable adapter metrics use only: `scheduler`, `lane`, `shard_id`, `action`, `reason`, `scope`, `quantile`.
- Raw job keys, request IDs, and idempotency keys are **not** exported as labels by default.
- `ForbiddenMetricLabelNames()` documents prohibited label names.

### New public API

| Symbol | Purpose |
|--------|---------|
| `ObservabilityStability`, `MetricDescriptor`, `LabelDescriptor` | Stability classification |
| `StableMetricDescriptors()` | Stable prometheus metric inventory |
| `ExperimentalMetricPatterns()` | Hook-adapter metric patterns |
| `AllowedDefaultMetricLabelNames()`, `ForbiddenMetricLabelNames()` | Label policy |
| `otel.StableTraceAttributeKeys()`, `otel.Attr*` | OTEL attribute contract |

### Hook panic behavior

User hooks run through `callHook`. Panics are recovered and do not stop workers. Recovered panics increment `HookPanicsRecovered()`. Hook payloads redact raw `Key` and `RequestID` by default; set `Observability.ExposeRawRequestIdentifiers` to opt in (warns with `KL_CONFIG_RAW_REQUEST_IDENTIFIERS_IN_HOOKS`).

### Debug snapshot

`DebugSnapshotVersion` is `"6"`. Field additions may occur in minors; version bumps are release-noted when programmatic consumers should react.

### Experimental metrics

Pipeline, continuation, and backend coordination counters remain **hook-adapter** responsibilities (see [metrics.md](../metrics.md)). They are listed in `ExperimentalMetricPatterns()` with `experimental` stability.
